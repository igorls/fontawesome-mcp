MCP Server Integration

Integrate FontAwesome icons directly into your development workflow with our Model Context Protocol server

VS Code Configuration

// .vscode/mcp.json
{
  "servers": {
    "fontawesome": {
      "command": "node",
      "args": ["./dist/server.js"],
      "env": {
        "FONTAWESOME_API_TOKEN": "your-token-here"
      }
    }
  }
}

Claude Desktop Configuration

// claude_desktop_config.json
{
  "mcpServers": {
    "fontawesome": {
      "command": "node",
      "args": ["path/to/dist/server.js"],
      "env": {
        "FONTAWESOME_API_TOKEN": "your-token-here"
      }
    }
  }
}

Available MCP Tools

search_icons

Search for FontAwesome icons with fuzzy matching

Parameters:
- query: Search terms
- limit: Max results (1-50)
- family_styles: Filter by families
- include_svgs: Include SVG data

get_icon_by_name

Get specific icon by exact name or alias

Parameters:
- name: Exact icon name
- family_styles: Filter by families
- include_svgs: Include SVG data

get_release_info

Get FontAwesome version and release details

Parameters:
- version: FA version (e.g., '7.x')
- list_all: List all releases

get_family_styles

List all available family/style combinations

Parameters:
- version: FA version (e.g., '7.x')

MCP Tool Insights & Improvements

Based on real-world testing, here are key insights about the MCP tools:

Search Strategy: Use single keywords instead of complex phrases. "star" works better than "star rocket crown plug"
Icon Coverage: Not all icons support all Pro+ families. Use get_icon_by_name to see available families per icon
Pro+ Discovery: Some icons like "rocket-launch" are exclusive to certain Pro+ families (whiteboard)
Performance: family_styles filtering can significantly improve search relevance
